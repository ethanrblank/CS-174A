~~~~~ READ ME

This file contains my code and a sample video for my bee scene. In order to make this code run:
1. unzip the file
2. drag the folder to your desktop
3. open up command prompt and set your current directory to cd C:\Users\yourname\Desktop\204381410
4. once you are at the correct directory, host it in your web browser by typing “python -m http.server” into command prompt
5. open your web browser and navigate to localhost:8000
6. open the template_demo.html
7. press Bee_Scene
8. Press alt+a to watch the scene
